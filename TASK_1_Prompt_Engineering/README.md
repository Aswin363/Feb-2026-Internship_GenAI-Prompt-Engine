🧠 GenAI Prompt Engineering – LangChain Prompt Templates
This repository contains the implementation of a Mini Prompt Engine built using LangChain PromptTemplate and ChatPromptTemplate.

The project focuses on designing dynamic, reusable, and structured prompt systems instead of hardcoded prompts, which is a key concept in real-world Generative AI applications.

📌 Objective
The objective of this assignment is to build a dynamic and reusable prompt system using LangChain.

The system:

Accepts user inputs (topic, audience, tone, style)
Validates inputs
Generates structured prompts dynamically
Demonstrates real-world prompt engineering pipelines
